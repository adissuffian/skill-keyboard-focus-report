# Quick Start

Use this skill in any repository by asking:

- Run keyboard-focus-visual-report for Home and Sign in on http://localhost:8080/home. Save to docs/home-keyboard-focus-report-<date>.html
- Rerun keyboard-focus-visual-report in strict clean start mode for Delivery first page only.
- Rerun keyboard-focus-visual-report for Delivery only from Home and correct any wrong outline labels based on visible evidence.
- Run keyboard-focus-visual-report Delivery-only correction rerun and replace stale screenshot paths with a new dated folder.

Recommended defaults:

- Wait 700ms after each Tab before screenshot
- Capture only visible and meaningful focus targets
- Preserve canonical section order
- Use Outline-only pills
- If a visible focus ring is shown, use Outline: visible focus ring (never none)
